# Security Risk Portfolio

This portfolio is a fictional security and GRC analysis for HarborPay, a small Boston-based fintech company.

The goal is to demonstrate beginner-level cybersecurity, GRC, IAM, secure coding, and AI-assisted risk analysis skills.

## Company

HarborPay is a fictional payments technology company with 120 employees. It uses cloud services, identity tools, code repositories, and third-party vendors to support customer payments.

## Portfolio Sections

1. Company overview
2. Asset inventory
3. Risk register
4. NIST/CIS control map
5. Vendor risk questionnaire
6. IAM login investigation
7. Secure Java RBAC app
8. AI GRC assistant
9. Executive summary

## Current Status

This repo is currently being built as a learning portfolio. The first phase focuses on understanding a fictional company's systems, identifying security risks, and documenting practical controls.
