# HarborPay Company Profile

## Company Summary

HarborPay is a fictional Boston-based fintech company that provides payment processing tools for small businesses.

## Business Model

HarborPay helps merchants accept online payments, track transactions, and manage customer billing.

## Company Size

- 120 employees
- Hybrid workforce
- Boston headquarters
- Engineering, sales, finance, customer support, and operations teams

## Core Technology

- Google Workspace for email and documents
- Okta for identity and single sign-on
- AWS for cloud hosting
- GitHub for source code
- Slack for internal communication
- Stripe for payment processing
- Jira for project tracking

## Sensitive Data

HarborPay may handle:

- Customer names
- Business emails
- Payment metadata
- Internal financial reports
- Source code
- Employee account information

## Initial Security Concerns

- Employee account compromise
- Weak access controls
- Overly broad cloud permissions
- Source code or secret exposure
- Vendor risk from payment and cloud providers
