# 08 - Report Templates

These templates keep every hands-on artifact specific, concise, and consistent. They are based on the structure I want to use in interviews: summary, actions taken, findings, decision, evidence, limitations, and references.

## Files

- [Triage note](triage-note-template.md): one alert, event, or short investigation.
- [Lab report](lab-report-template.md): setup, test, troubleshooting, and results for a technical build.
- [Evidence log](evidence-log-template.md): sanitized evidence inventory tied to report claims.

## Rule

Do not check a box or remove a prompt until the related action actually happened. A clean template is not evidence of completed technical work.
